# PEAEDGE

PeaEdge是一个物联网边缘节点软件，可以运行在各种支持Linux的嵌入式设备上，如Raspberry Pi。

[EN](README.md)

## 特点

- [x] 标准Modbus设备访问管理
- 支持Mqtt协议
- 集成的LoraWan协议支持
- 蓝牙终端访问支持
- Zigbee协议支持
- [x] Hj212协议支持
- 摄像头接入支持(Openonvif)
